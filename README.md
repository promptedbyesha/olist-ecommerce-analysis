Olist E-Commerce Funnel & Retention Analysis
Objective
Identify purchase funnel drop-off, cohort retention patterns, and revenue concentration across 99,441 real Brazilian e-commerce orders.

Tools
SQL (MySQL) | Python (pandas, seaborn) | Tableau Public

Key Findings
1.79% shipping-to-delivery drop-off = ~1,200 lost orders at scale
Month-2 cohort retention ~30–35%; heavy dependence on new acquisition
Bed & Bath (R$1.71M) and Health & Beauty (R$1.66M) drive 40%+ of GMV
Dashboard
View on Tableau Public

Resume Bullet
Analyzed 100K+ real e-commerce orders (Olist dataset) using SQL and Python to map purchase funnel drop-off across 4 stages; built Tableau cohort retention dashboard revealing ~30–35% month-2 churn; prioritized top 3 fixes using impact vs effort framework.
